Jeśli chesz uruchomić stronę lokalnie:
1. Przejdź do folderu w koncoli cd...
2. Wpisz polecenie npm install.
2. Następnie wpisz "gulp" zatwierdź enterem.
3. Uruchomi się w domyślnej przeglądarce strona.

Jeśli chcesz stworzyć wersje produkcyjną:
1. Wpisz do konsoli "gulp build"
2. stworzy się folder dist z plikami zminifikowanymi
